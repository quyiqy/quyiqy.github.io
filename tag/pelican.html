    让我们开始吧

    你所看到的博客，是用Pelican搭建的。 Pelican是一套开源的使用Python编写的博客静态生成工具，它的特点如下： - 可以添加文章和和创建页面 - 可以使用MarkDown reStructuredText 和 AsiiDoc 的格式来书写 - 同时使用 Disqus评论系统, 支持 RSS和Atom输出, 插件, 主题, 代码高亮等功能 - 采用Jajin2模板引擎, 可以很容易的更改模板.

    在搭建微博之前，建议先学习:git基本教程

    起码你需要掌握以下知识

    1. 在github中创建一个repo：yourname.github.io
    2. git init #初始化git
    3. git add . #将工作区内容推送到暂存区
    4. git commit -m '你修改的内容' #提交暂存区内容
    5. git remote add origin http://github.com ...
